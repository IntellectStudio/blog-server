# Express.js App

## 介绍
本项目使用Node.js + Express.js 搭建

## 路径
``localhost:3000/docs`` => 文档页面

## 常用命令
````
yarn build        // 启动Node.js 服务器
yarn dev          // Node.js 本地开发
yarn docs:dev     // 文档写作模式
yarn docs:build   // 文档打包
yarn compile:prod // 生产部署
````
